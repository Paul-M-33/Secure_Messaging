# Secure Messaging App

Cette application permet de discuter de manière sécurisée entre plusieurs utilisateurs grâce au chiffrement AES/RSA.

## Prérequis

- Python 3.10+ recommandé
- Installer les dépendances :

pip install -r requirements.txt

## Lancer le serveur

python network/server.py

## Lancer les clients

python main.py

Chaque exécution de main.py ouvre une nouvelle fenêtre pour un utilisateur.

Vous pouvez lancer autant de clients que vous le souhaitez pour discuter entre eux.
